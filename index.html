А что это вообще такое?

Что умеет?

  • Хранить векторные и растровые данные
  • Подключать внешние PostGIS базы данных
  • Выступать в роли сервера WMS и WFS-T (и клиента WMS)
  • Отрисовывать данные с помощью QGIS и MapServer
  • Управлять правами пользователей
  • Публиковать карты
  • HTTP API

Как попробовать?

  1. Создать Веб ГИС в облаке nextgis.ru
  2. Развернуть у себя с нашей помощью (On-Premises)
  3. Разобраться с установкой самостоятельно, код открыт

Собственно, что нового (часть 1)

  • Современный вид панели управления
  • Сортировка ресурсов
  • Упрощённый вид карты для вставки через iframe
  • Подсистема управления настройками
  • CORS (Cross-origin resource sharing)
  • WMS-клиент 1.3.0

Перечисление нововведений указано в хронологическом порядке

Собственно, что нового (часть 2)

  • Контрол печати карты
  • Поддержка одноканальных индексированных растров
  • "Шторка"
  • Редактирование слоёв PostGIS
  • WMS GetLegendGraphic
  • Редактирование геометрий
  • Векторные тайлы
